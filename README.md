# AnimaI T2I

画像生成AIのプロンプトをクリックだけで簡単に構築し、コピーできるツールです。
CSVファイルを編集することで、独自のタグを自由に追加・管理できます。

## 起動方法

AnimaI.zipをダウンロードして、exeを起動してください。

## プロンプトの追加
csvファイルを編集して、プロンプトの追加を行えます。
日本語名|prompt

## 使用技術
- **Rust** (Axum, RustEmbed)
- **HTML/CSS/JS** (Vanilla, API fetching)

# AnimaI T2I macOS HTML版

このフォルダは、macOSユーザー向けのHTML単体版です。RustサーバーやWindows用 `animai.exe` は不要です。

## 起動方法

1. `macos` フォルダをMacへコピーします。
2. `index.html` をダブルクリックしてブラウザで開きます。
3. タグをクリックしてプロンプトを作成し、コピーして画像生成ツールへ貼り付けます。

## お気に入りについて

- お気に入りはブラウザの `localStorage` に保存されます。
- Safari、Chrome、Firefoxなど、ブラウザごとに保存先は分かれます。
- 別のMacや別ブラウザへ移す場合は「エクスポート」でJSONを書き出し、移行先で「インポート」を使ってください。
- プライベートブラウズでは保存内容が残らない場合があります。

## 注意点

- HTML単体版のため、CSVフォルダを自動で読み直す機能はありません。
- 同梱データを更新する場合は、Windows側で `csv` の内容から `prompts-data.js` を再生成してください。
- 任意フォルダへの直接保存やフォルダを開く機能は、ブラウザのセキュリティ制約により搭載していません。
